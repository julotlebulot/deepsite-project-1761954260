Terminal d'attaque

Bienvenue dans le HackLab Terminal.
Ce simulateur vous permet de pratiquer des commandes de pentest dans un environnement sécurisé.

Commandes disponibles:
- nmap [cible]
- gobuster [mode] [options]
- help
- clear

Essayez: nmap hacklab.local
hacker@hacklab:~$ _

Astuce Nmap

Pour scanner une cible: nmap -sV -A hacklab.local
-sV: Détection des versions
-A: Scan agressif

Astuce Gobuster

Pour fuzzer des répertoires: gobuster dir -u http://hacklab.local -w wordlist.txt
-u: URL cible
-w: Wordlist

Attention

Ce simulateur est pour l'apprentissage éthique uniquement.
Ne pas utiliser ces techniques sans autorisation écrite.

Challenges

Découverte du réseau

Difficulté: ★★☆☆☆

Utilisez Nmap pour découvrir tous les services actifs sur le réseau HackLab. Trouvez le port caché qui exécute un service FTP.

Nmap • Scanning réseau

Fuzzing de répertoires

Difficulté: ★★★☆☆

Utilisez Gobuster pour trouver le répertoire caché contenant les informations sensibles. Le répertoire commence par "admin".

Gobuster • Fuzzing web

Injection SQL

Difficulté: ★★★★☆

Exploitez la vulnérabilité SQL dans le formulaire de recherche pour obtenir les identifiants admin. La table s'appelle "users".

Web • Injection SQL

Cross-Site Scripting

Difficulté: ★★★☆☆

Trouvez et exploitez la vulnérabilité XSS pour exécuter du JavaScript arbitraire. Le vecteur est dans le paramètre "search".

Web • XSS
